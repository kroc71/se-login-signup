# Login and Signup Web Pages

## Project Description
This project is part of the ICSI 418Y Software Engineering course.  
The objective of the assignment is to build basic front-end web pages using HTML and CSS and manage the project using Git and GitHub.

The project includes two pages:
- Login Page
- Signup Page

Both pages are styled using a shared CSS file to maintain a consistent design.

## Project Structure

## Login Page Features

- Field for **User ID**
- Field for **Password**
- Submit button
- Link to the Signup page

## Signup Page Features

- Field for **First Name**
- Field for **Last Name**
- Field for **User ID**
- Field for **Password**
- Submit button
- Link to the Login page

## Styling

The project uses a single CSS file called `styles.css` to ensure a consistent layout across both pages.

The form is centered on the page and includes clean spacing and alignment.

No external frameworks were used.

## Technologies Used

- HTML
- CSS
- Git
- GitHub
- VS Code
